## 🩸 O QUE É ISTO
Isto não é um repositório. **Isto é um terreiro digital.**
Uma experiência cinematográfica imersiva que pulsa na fronteira entre o código e o sagrado — onde a tecnologia se ajoelha diante da ancestralidade das **Pombagiras**.
Uma mulher de vestido negro caminha entre mausoléus de mármore à meia-noite. Ela fala em português brasileiro — sobre **responsabilidade** ao invocar forças que a maioria sequer compreende. No fundo, estátuas de anjos rachados e demônios distorcidos observam. Pétalas de rosa sopram como sangue no vento. Chamas de tridente iluminam o que deveria ficar no escuro.
E então elas se manifestam:
- 🌹💀 **Pombagira Rosa Caveira** — sobreposição espectral rosada, coroa de rosas, crânio luminoso
- ⛓️👻 **Pombagira Mulambo** — espectro púrpura, panos rasgados, correntes que rangem

## ⛧ COMO USAR
```bash
# É um arquivo HTML único. Não precisa de nada.
# Abra no navegador. Entre na encruzilhada. Sinta.
# Opção 1: Abrir diretamente
open index.html
# Opção 2: Servir localmente
npx serve .
# Opção 3: Colocar no GitHub Pages
# Apenas faça push do index.html e ative Pages
```
> **⚠️ AVISO:** Use tela cheia para a experiência completa. A encruzilhada não cabe em janela pequena.
